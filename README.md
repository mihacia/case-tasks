# Кейс-задача № 3: Сумма отрицательных элементов между максимумом и минимумом

## 📌 О проекте

Данный репозиторий содержит решение кейс-задачи, выполненной в рамках **учебной практики** по направлению «Информационная безопасность».

**Задача:**  
Дан одномерный массив `A` размерности `N`. Необходимо найти сумму отрицательных элементов, расположенных **между** максимальным и минимальным элементами массива.

Решение реализовано на языке программирования:
- Python

---

## ⚙️ Что делает программа

Программа выполняет следующие шаги:

1. **Принимает на вход** одномерный массив чисел (целых или вещественных).
2. **Находит** позицию (индекс) первого максимального элемента и первого минимального элемента.
3. **Определяет** диапазон индексов между ними (исключая сами экстремумы).
4. **Вычисляет** сумму всех отрицательных элементов, попавших в этот диапазон.
5. **Выводит** результат и промежуточные данные для наглядности.

# Кейс-задача № 4: Максимальная сила драконьей стаи

**Задача:**  
Известно, что у дракона может быть несколько голов (максимум 7) и его сила определяется числом голов. Сила стаи драконов вычисляется как **произведение** значений числа голов каждого из драконов. Дано суммарное количество голов `N`. Требуется найти **максимально возможное значение силы** этой стаи.

**Пример:**  
Если в стае 3 дракона с 3, 4 и 5 головами соответственно, то сила равна 3 × 4 × 5 = 60.  
При N = 12 оптимальное разбиение: 3 + 3 + 3 + 3 = 12 → сила = 3 × 3 × 3 × 3 = 81.

Решение реализовано на языке программирования:
- Python

---

## ⚙️ Что делает программа

Программа выполняет следующие шаги:

1. **Принимает на вход** натуральное число `N` (1 ≤ N ≤ 99) — общее количество голов драконьей стаи.
2. **Анализирует** возможность разбиения числа `N` на слагаемые от 1 до 7 (количество голов у каждого дракона).
3. **Находит оптимальное разбиение**, максимизирующее произведение.
4. **Вычисляет** максимальную силу стаи.
5. **Выводит** результат и демонстрирует оптимальное разбиение с проверкой.

**Ключевая логика:**
- Для максимизации произведения используется **число 3** как оптимальный множитель.
- Если остаток от деления на 3 равен 1, заменяем `3+1` на `2+2` (так как 3×1 < 2×2).
- Если остаток равен 2, добавляем одну двойку.
